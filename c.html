C++ COURSE

C/C++ course at coding house. C is a programming language developed by Dennis Ritchie in 1972. C is a very popular programming language. C is commonly used to create desktop applications. Even the Linux operating system was developed using the C language. C++ is an extension of C by adding the concept of OOP (Object Oriented Programming). C can also be said to be the Queen of the Programming Language because many programming languages ​​have adopted C concepts and syntax, including Java, Python, PHP and others. That's why a programmer who masters C can easily learn other programming languages.


Target Participants : To take part in the C++ Home Coding Course, participants are not required to have previous coding knowledge because the discussion in this course starts from the basics so that anyone who is new to the world of coding can follow it.
