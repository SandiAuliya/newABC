WEB DESIGN COURSE

Learning websites starts with Web Design. Let's take a web design course at the ABC Learning Center. Web Design is the process of creating a website, this includes several different aspects, including: website page layout, content, and graphic design. In this course, you will learn how to create a web page. You will learn basic knowledge about websites, from HTML, CSS to Javascript. Then you will also learn how to make responsive website pages using Bootstrap.


This course is the most comprehensive course covering all five programming: HTML, CSS, Javascript, PHP, and MySQL using Indonesian on Udemy. It doesn't matter whether you have ever had a website or have never done programming before, this course is definitely for you. Each language or program script has specific functions, namely:
HTML: to design the structure of website pages. The entire website is built from HTML structure.
CSS: to set how the website will look on the monitor screen. Whether or not the appearance of the website is set using CSS.
Javscript: to create interactive websites. Javascript is a client-side based programming script so it is often used to quickly manage interactions between users and websites.
PHP: to make the website more powerful because it can handle logic and database storage. Most of the websites in this world are built by PHP.
MySQL: to store data entered by the user into the database system.


Target Participants : To take part in this course, participants are not required to have prior coding knowledge. can be followed by anyone who wants to have basic skills in designing modern and responsive website pages. because the discussion in this course starts from the basics so that anyone can follow it, including those who are unfamiliar with the world of coding. You will learn how to create a website using the Bootstrap framework.
